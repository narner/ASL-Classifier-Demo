The object detector model, located at ASL Classifier Demo/ASL Classifier.mlmodel, is trained on David Lee's American Sign Language Letters Dataset, hosted on Roboflow here: https://public.roboflow.com/object-detection/american-sign-language-letters

The source code is based on a modified version of Apple's Vision example project Breakfast Finder, available here: https://developer.apple.com/documentation/vision/recognizing_objects_in_live_capture
